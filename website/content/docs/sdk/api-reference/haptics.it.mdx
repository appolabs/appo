## Utilizzo

### Feedback di Impatto

```typescript
appo.haptics.impact('light');  // 'light' | 'medium' | 'heavy'
```

### Feedback di Notifica

```typescript
appo.haptics.notification('success'); // 'success' | 'warning' | 'error'
```
