## Usage

### Check Availability

```typescript
const available = await appo.biometrics.isAvailable();
```

### Authenticate

```typescript
if (available) {
  const success = await appo.biometrics.authenticate('Confirm your identity');
  if (success) {
    // User authenticated
  }
}
```

## Browser Fallback

| Method | Fallback |
|--------|----------|
| `isAvailable()` | Returns `false` |
| `authenticate()` | Returns `false` |
