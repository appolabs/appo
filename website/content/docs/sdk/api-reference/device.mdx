## Usage

```typescript
const info = await appo.device.getInfo();
console.log(info.platform);    // 'ios' | 'android'
console.log(info.osVersion);   // e.g. '17.4'
console.log(info.appVersion);  // e.g. '1.2.0'
console.log(info.deviceId);    // Unique device identifier
console.log(info.deviceName);  // e.g. 'iPhone 15 Pro'
console.log(info.isTablet);    // true | false
```
