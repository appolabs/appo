## Utilizzo

```typescript
const result = await appo.share.open({
  title: 'Guarda questo',
  message: 'Contenuto da condividere',
  url: 'https://example.com',
});

if (result.success) {
  console.log('Condiviso tramite:', result.action);
}
```
