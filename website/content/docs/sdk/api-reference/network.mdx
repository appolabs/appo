## Usage

### Get Current Status

```typescript
const status = await appo.network.getStatus();
console.log(status.isConnected, status.type);
// type: 'wifi' | 'cellular' | 'none' | 'unknown'
```

### Subscribe to Changes

Listens on the `network.change` event channel.

```typescript
const unsubscribe = appo.network.onChange((status) => {
  console.log('Network changed:', status.isConnected, status.type);
});

// Stop listening
unsubscribe();
```

| Method | Fallback |
|--------|----------|
