## Utilizzo

### Richiesta Permesso

```typescript
const status = await appo.camera.requestPermission();
// stato: 'granted' | 'denied' | 'undetermined'
```

### Scattare una Foto

```typescript
if (status === 'granted') {
  const result = await appo.camera.takePicture();
  console.log(result.uri, result.width, result.height);
  // result.base64 e disponibile se il gestore nativo lo fornisce
}
```
