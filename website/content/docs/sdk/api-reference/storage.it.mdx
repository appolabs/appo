## Utilizzo

### Salvare un Valore

```typescript
await appo.storage.set('auth_token', 'abc123');
```

### Recuperare un Valore

```typescript
const token = await appo.storage.get('auth_token');
// token: string | null
```

### Eliminare un Valore

```typescript
await appo.storage.delete('auth_token');
```

## Fallback nel Browser

Tutti i metodi di archiviazione utilizzano `localStorage` quando l'esecuzione avviene al di fuori di un contenitore nativo.

| Metodo | Fallback |
|--------|----------|
| `get()` | `localStorage.getItem()` |
| `set()` | `localStorage.setItem()` |
| `delete()` | `localStorage.removeItem()` |
