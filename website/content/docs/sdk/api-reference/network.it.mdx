## Utilizzo

### Ottenere lo Stato Attuale

```typescript
const status = await appo.network.getStatus();
console.log(status.isConnected, status.type);
// tipo: 'wifi' | 'cellular' | 'none' | 'unknown'
```

### Sottoscriversi ai Cambiamenti

Ascolta sul canale evento `network.change`.

```typescript
const unsubscribe = appo.network.onChange((status) => {
  console.log('Rete cambiata:', status.isConnected, status.type);
});

// Smetti di ascoltare
unsubscribe();
```

## Fallback nel Browser

| Metodo | Fallback |
|--------|----------|
| `getStatus()` | Restituisce `{ isConnected: navigator.onLine, type: 'unknown' }` |
| `onChange()` | Ascolta gli eventi `online`/`offline` del browser |
