## Utilizzo

```typescript
const info = await appo.device.getInfo();
console.log(info.platform);    // 'ios' | 'android'
console.log(info.osVersion);   // es. '17.4'
console.log(info.appVersion);  // es. '1.2.0'
console.log(info.deviceId);    // Identificativo univoco del dispositivo
console.log(info.deviceName);  // es. 'iPhone 15 Pro'
console.log(info.isTablet);    // true | false
```
