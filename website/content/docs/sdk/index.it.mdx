L'SDK `@appolabs/appo` fornisce un'API JavaScript unificata per accedere alle funzionalità native del dispositivo da applicazioni web eseguite all'interno di WebView React Native.

## Cosa Fornisce l'SDK

- **9 API di funzionalità** — Notifiche push, biometria, fotocamera, posizione, feedback aptico, archiviazione, condivisione, rete e informazioni dispositivo
- **Protocollo bridge** — Comunicazione richiesta/risposta con il layer nativo tramite `postMessage`
- **Sistema eventi** — Sottoscrizione agli eventi nativi (messaggi push, cambiamenti di rete)
- **Fallback browser** — Tutte le API restituiscono valori predefiniti ragionevoli al di fuori di un container nativo
- **Sicurezza dei tipi** — TypeScript completo con interfacce esportate
